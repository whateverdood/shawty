Overview
========
You've downloaded Shawty, a Groovy/Java based XPath text extractor.  For 
information on how to use Shawty's XPathExtractor see 
src/test/groovy/com/googlecode/shawty/XPathExtractorTests.groovy

Dependencies
============
Java 5+, Groovy 1.6+ and Maven 3

Build
=====
To build Shawty, install Maven 3 (http://maven.apache.org) and run:

$ mvn clean package
